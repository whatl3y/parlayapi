Parlay API

Get the true odds of a single sports wager (given american odds) or a set of odds assuming they will be placed together in a parlay.

Examples

  1. Single Event (-110 American Odds)
    GET https://www.parlayapi.com/-110
    JSON Response:
                      
{
  "american": "-110",
  "decimal": "1.91",
  "fractional": "9091/10000",
  "general_for": "1.91 for 1",
  "general_to": "0.91 to 1",
  "odds": "-110",
  "wager100_totalwin": "190.91",
  "wager100_win": "90.91"
}
  2. 2-Team Parlay Example
    GET https://www.parlayapi.com/-110/-150
    JSON Response:
                      
{
  "american": "+218",
  "decimal": "3.18",
  "fractional": "10909/5000",
  "general_for": "3.18 for 1",
  "general_to": "2.18 to 1",
  "odds": "-110,-150",
  "wager100_totalwin": "318.18",
  "wager100_win": "218.18"
}
  3. 3-Team Parlay Example
    GET https://www.parlayapi.com/-110/175/-210
    JSON Response:
                      
{
  "american": "+675",
  "decimal": "7.75",
  "fractional": "27/4",
  "general_for": "7.75 for 1",
  "general_to": "6.75 to 1",
  "odds": "-110,175,-210",
  "wager100_totalwin": "775.00",
  "wager100_win": "675.00"
}
  4. 8-Team Parlay Example
    GET https://www.parlayapi.com/-110/175/-125/135/-110/-115/-275/190
    JSON Response:
                      
{
  "american": "+31245",
  "decimal": "313.45",
  "fractional": "624893/2000",
  "general_for": "313.45 for 1",
  "general_to": "312.45 to 1",
  "odds": "-110,175,-125,135,-110,-115,-275,190",
  "wager100_totalwin": "31344.65",
  "wager100_win": "31244.65"
}
